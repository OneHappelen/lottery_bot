# Telegram Giveaway Bot

## Описание
Этот проект представляет собой Telegram-бота для автоматического мониторинга заданных каналов с целью поиска розыгрышей и отправки их в целевой канал. Также бот может извлекать дату из сообщений и составлять отчеты.

## Функционал
- Мониторинг каналов для поиска розыгрышей.
- Пересылка сообщений с розыгрышами в целевой канал.
- Извлечение даты из сообщений и отправка в отчетный канал.
- Очистка базы данных от старых записей.
- Ежедневный отчет о найденных розыгрышах.


## Структура проекта
- **main.py**: Основной файл для запуска бота.
- **config.py**: Конфигурационные данные (группы, каналы, токены).
- **database.py**: Работа с базой данных SQLite.
- **bot.py**: Основная логика работы с Telegram API.
- **utils.py**: Вспомогательные функции.

## Авторы
Этот проект разработан для автоматизации работы с Telegram-каналами.
